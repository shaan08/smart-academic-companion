# MentorMate AI

Your Smart Academic Companion - AI-powered tools for studying, career planning, and academic success.

## Features

- **AI Study Assistant**: Get instant answers, summaries, and personalized study materials
- **Career Planning**: AI-powered career guidance and planning tools
- **Academic Excellence**: Tools designed to elevate your learning journey

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (Backend)

## Getting Started

### Prerequisites

- Node.js & npm installed - [Install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to project directory
cd mentormate-ai

# Install dependencies
npm install

# Start development server
npm run dev
```

## Development

The app will be available at `http://localhost:5173`

## Building for Production

```sh
npm run build
```
